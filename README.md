# JWTpa

A robust asynchronous teleport request plugin for Endstone.

## Features
- `/tpa` - Request teleport to a player.
- `/tpahere` - Request player to teleport to you.
- `/tpaccept` - Accept teleport request.
- `/tpdeny` - Decline teleport request.
- `/tpcancel` - Cancel your teleport request.

## Commands
- `/tpa <player: string>`
- `/tpahere <player: string>`
- `/tpaccept` (aliases: `/tpaaccept`, `/tpac`)
- `/tpdeny` (aliases: `/tpadeny`, `/tpad`)
- `/tpcancel`
